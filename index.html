Passos para Hospedar um Projeto no GitHub Pages

  1. 1. Crie um repositório no GitHub com o mesmo nome do seu projeto
  2. 2. No terminal, inicialize o repositório local git init
  3. 3. Altere o nome da branch principal para main
  4. 4. Adicione arquivos com git add . e faça commit com git commit -m first commit
  5. 5. Conecte ao repositorio remoto com: git remote add origin https://github.com/SEU_USUARIO/NOME_DO_REPOSITORIO
git remote origin https://github.com/html.git
